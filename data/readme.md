# Data Folder

This folder contains motorcycle crash data used for the ANA 699 capstone project.

## Data Source
All data files were obtained from the Transportation Injury Mapping System (TIMS), developed by the UC Berkeley Safe Transportation Research and Education Center (SafeTREC). The data are derived from Californiaâ€™s Statewide Integrated Traffic Records System (SWITRS), which contains official, police-reported traffic collision records from the California Highway Patrol and local law enforcement agencies.

## Files

- LA_County.csv  
  Motorcycle crash records for Los Angeles County.

- RIV_County.csv  
  Motorcycle crash records for Riverside County.

- SB_County.csv  
  Motorcycle crash records for San Bernardino County.

- SD_County.csv  
  Motorcycle crash records for San Diego County.

- SoCal_Motorcycle_Crashes.csv  
  Combined dataset containing motorcycle crash records from all four counties listed above. Each row represents a single motorcycle crash event and includes injury severity, alcohol involvement, roadway and environmental conditions, temporal variables, and geocoded location data.

## Notes
The combined dataset is intended to be used for analysis and modeling. The individual county files are retained for reference and data verification purposes.
